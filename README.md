🎬 Amazon Prime Video Clone

A modern and responsive front-end clone of the Amazon Prime Video platform built using HTML, CSS, and JavaScript. This project recreates the core visual design and user experience of Amazon Prime Video, providing users with an engaging streaming-platform interface while demonstrating essential front-end development skills.

📖 Overview

The goal of this project is to replicate the layout, styling, and overall appearance of the Amazon Prime Video website. It focuses on creating a clean and interactive user interface that closely resembles the original platform while maintaining responsiveness across different screen sizes.

This clone includes key UI components such as a navigation bar, hero section, content categories, movie cards, and footer. The project serves as a practical exercise in modern web development and showcases the implementation of responsive design principles, reusable layouts, and interactive elements.

✨ Features
Responsive and user-friendly interface
Modern navigation bar with menu options
Attractive hero/banner section
Multiple content categories
Interactive movie and TV show cards
Smooth hover effects and transitions
Structured footer section
Mobile, tablet, and desktop compatibility
Clean and organized code structure
🛠️ Technologies Used
HTML5 – Structure and content
CSS3 – Styling and responsiveness
JavaScript – Basic interactivity
Flexbox & CSS Grid – Layout management
