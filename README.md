A responsive front-end clone of the Amazon Prime Video homepage built using HTML, CSS, and JavaScript. This project replicates the basic user interface of Amazon Prime, including the navigation bar, hero section, content categories, movie/show cards, and footer. The focus of this project is on practicing web design, layout structuring, responsive styling, and creating a visually appealing streaming platform interface. This clone is intended for educational purposes and does not include backend functionality, authentication, video streaming, or database integration.
